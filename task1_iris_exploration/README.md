# Task 1: Iris Dataset Exploration

## Objective
Load, inspect, and visualize the Iris dataset to understand feature relationships and distributions.

## Files
- `task1_solution.py`: pandas inspection + seaborn/matplotlib plots saved to `plots/`
- `app.py`: Streamlit frontend for interactive inspection and visualization
- `data/iris.csv`: local dataset copy created automatically

## Run
```bash
python task1_solution.py
streamlit run app.py
```
