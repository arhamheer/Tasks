# Task 2: Short-Term Stock Price Prediction

## Objective
Predict the next day's close price using Open, High, Low, and Volume.

## Files
- `task2_solution.py`: downloads stock data from Yahoo Finance and trains a model
- `app.py`: Streamlit interface to choose ticker/model and visualize results
- `data/<ticker>_historical.csv`: auto-saved historical data

## Run
```bash
python task2_solution.py
streamlit run app.py
```
