# Task 3: Heart Disease Prediction

## Objective
Predict heart disease risk with a binary classification model.

## Files
- `task3_solution.py`: data loading, cleaning, EDA, model training, and evaluation
- `app.py`: Streamlit dashboard for model metrics and manual prediction
- `data/heart_disease.csv`: auto-saved cleaned data

## Run
```bash
python task3_solution.py
streamlit run app.py
```
