# Task 6: House Price Prediction

## Objective
Predict property sale prices from housing features.

## Files
- `task6_solution.py`: preprocessing, training, evaluation (MAE/RMSE), and plot
- `app.py`: Streamlit UI for metrics and prediction comparison chart
- `data/house_prices.csv`: auto-saved dataset

## Run
```bash
python task6_solution.py
streamlit run app.py
```
